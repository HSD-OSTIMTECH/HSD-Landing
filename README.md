## Kurulum

1. Depoyu klonlayın:
   ```bash
   git clone https://github.com/HSD-OSTIMTECH/HSD-Frontend.git
   cd HSD-Frontend
   ```
2. Bağımlılıkları yükleyin:
   ```bash
   pnpm install
   # veya npm install
   # veya yarn install
   ```
3. Geliştirme sunucusunu başlatın:
   ```bash
   pnpm dev
   # veya npm run dev
   # veya yarn dev
   ```
4. [http://localhost:3000](http://localhost:3000) adresinde görüntüleyin.
